## 🇷🇺 Инструкция по запуску (Russian)

### Условия 
* На компьютере должен быть установлен **Node.js**.
* Должен быть установлен и запущен **MongoDB Compass**.

### Как запустить:
1. **Запустите командную строку в самой папке с проектом.** Например:
   * Встроенный терминал **VS Code** (нажмите "Ctrl + ` ")
   * **Git Bash** (нажмите правой кнопкой мыши внутри папки -> *Git Bash Here*)
   * **Windows Shell** (Командная строка или PowerShell)

2. **Установите зависимости:** Введите команду в терминале и дождитесь окончания:
   ```bash
   npm install

3. **Запустите проект:** Выполните команду для старта:
   ```bash
   npm run

## 🇬🇧 Startup Guide (English)

### Requirements

* **Node.js** must be installed on your computer.
* **MongoDB Compass** must be installed and running.

### How to run:

1. **Open the command line directly in the project folder.** For example:
* **VS Code** integrated terminal (press "Ctrl + ` ")
* **Git Bash** (right-click inside the folder -> *Git Bash Here*)
* **Windows Shell** (Command Prompt or PowerShell)


2. **Install dependencies:** Run the following command and wait for it to finish:
```bash
npm install

```


3. **Start the project:** Run the start command:
```bash
npm run
